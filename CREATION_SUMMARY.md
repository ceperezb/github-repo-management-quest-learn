## Intentional Design Choices

### 1. Focus on Markdown/Docs (Not Code)
- Primary focus is documentation and content management
- Small amount of Python code for realism and utility
- Emphasizes skills relevant to content developers

### 2. AI as Force Multiplier
- AI tools integrated throughout
- Specific prompts provided as learning aids
- Emphasis on validating AI outputs
- Human judgment still central

### 3. Hands-On Learning
- Learners produce actual deliverables
- No multiple-choice tests
- Real decision-making required
- Authentic tasks, not toy problems

### 4. Scalable and Extensible
- Clear structure for adding content
- Template-based approach
- Easy to customize for specific teams
- Can add scenarios or tasks

### 5. Self-Paced Format
- Can complete in one session or multiple
- Clear time estimates for planning
- Success criteria for self-assessment
- Optional solution guides for validation
